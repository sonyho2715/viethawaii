# AI Prompt Engineer

Expert in crafting effective prompts and integrating AI into workflows.

## Prompt Engineering Principles

**Be Specific:**
❌ "Write about marketing"
✅ "Write a 500-word blog post about email marketing for SaaS startups"

**Provide Context:**
```
You are a senior marketing strategist.
Your audience is small business owners.
Write in a friendly, accessible tone.
```

**Give Examples:**
"Here's the format I want:
- Problem
- Solution
- Benefit

Now create 3 more like this..."

**Use Structured Prompts:**
```
Task: [What to do]
Context: [Background info]
Format: [Output structure]
Constraints: [Limitations]
```

## Prompt Templates

**Content Creation:**
```
Create a [type] about [topic]
For [audience]
Tone: [professional/casual/friendly]
Length: [words/characters]
Include: [specific elements]
```

**Analysis:**
```
Analyze [content/data]
Focus on: [aspects]
Provide: [insights/recommendations]
Format: [structure]
```

**Code Generation:**
```
Write [language] code that:
- [Function 1]
- [Function 2]
Include error handling and comments
```

## Chatbot Design

**Conversation Flow:**
1. Greeting

3. Gather required info
4. Provide solution
5. Confirm satisfaction

**Personality:**
